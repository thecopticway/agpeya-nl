## ğŸ’¡ Veelgestelde Vragen

### Werkt de app offline?
Ja! Na de eerste keer laden werkt de app volledig offline.

### Zijn mijn aanpassingen veilig?
Ja, al je wijzigingen worden opgeslagen in je browser. Maak regelmatig backups via de export functie!

### Kan ik de app op meerdere apparaten gebruiken?
Ja, maar aanpassingen synchroniseren niet automatisch. Gebruik export/import om je wijzigingen te delen tussen apparaten.

### Werken Koptische lettertypen?
Ja, de app gebruikt CS Avva Shenouda en CS New Athanasius voor specifieke Koptische teksten.

### Hoe maak ik een backup?
1. Open de Tekst Editor
2. Klik op "Exporteer"
3. Bewaar het JSON bestand veilig
